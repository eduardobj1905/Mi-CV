# Mi-CV
Stack Tecnológico de la Aplicación

Framework: Vue.js  

Lenguajes: JavaScript , HTML y CSS para la estructura y el diseño.

Control de Versiones: Git y GitHub 

Iniacializacion
Para ejecutar este proyecto en tu ordenador, sigue estos pasos:

Clonar el repositorio:
git clone https://github.com/eduardobj1905/Mi-CV.git

Instalar las dependencias:
Este paso descarga Vue, Vite y todas las librerías necesarias.
npm install

Lanzar el servidor de desarrollo:
Para ver los cambios en tiempo real en http://localhost:5173/.
npm run dev

Compilar para producción:
npm run build


Este proyecto no usa una base de datos
